## はじめに

PhotoUploader(以下「本アプリ」)は、iPhone 内の写真を **利用者ご自身が用意した Amazon Web Services(AWS)アカウント** にバックアップするためのアプリです。

本アプリの最大の特徴は、**写真データもアカウント情報も、本アプリの提供者のサーバーには一切送信・保存されない** ことです。すべてのデータは、利用者ご自身が管理する AWS 環境にのみ保存されます。本ポリシーは、本アプリがどのような情報をどのように扱うかを説明します。

## 提供者が収集・保存しない情報

本アプリの提供者は、以下の情報を **収集も保存もしません**。

- アップロードした写真、およびそのメタデータ(撮影日時・位置情報など)
- メールアドレス、パスワードなどのアカウント情報
- どの写真をいつアップロードしたかの記録

これらはすべて、利用者ご自身の AWS 環境内(後述)に保存され、提供者はアクセスできません。

## 本アプリが扱う情報と保存先

### 1. 写真とそのメタデータ

利用者が選択した写真は、利用者ご自身の AWS の Amazon S3(ストレージ)に直接アップロードされます。写真データは提供者のサーバーを経由しません。

### 2. アカウント情報(メールアドレス・パスワード)

本アプリのログインには、利用者ご自身の AWS 環境内に構築された Amazon Cognito(認証サービス)を使用します。メールアドレスとパスワードは Cognito が管理し、認証用のトークンは端末内の Keychain(iOS の暗号化された安全な保管領域)に保存されます。

### 3. カメラ

初回設定時に接続先情報の QR コードを読み取る目的でのみカメラを使用します。撮影した画像が保存・送信されることはありません。

### 4. 写真ライブラリへのアクセス

アップロードのために、利用者が iOS 標準の写真選択画面で **明示的に選んだ写真のみ** が本アプリに渡されます。本アプリが写真ライブラリ全体を自動的に読み取ることはありません。

### 5. 端末内に保存される設定・履歴

接続先設定、アップロード履歴(操作日時・枚数)、アップロード済み写真の識別情報は、端末内(iOS の UserDefaults)にのみ保存されます。本アプリを削除すると、これらの情報も削除されます。

## 第三者サービス

### Google AdMob(広告配信)

本アプリは、収益化のために Google の広告サービス「AdMob」を利用しており、広告の配信・測定のために Google が情報を収集する場合があります。収集され得る情報には、広告識別子、デバイス情報、おおよその位置情報、広告の表示・操作に関するデータが含まれることがあります。

- Google のプライバシーポリシー: https://policies.google.com/privacy
- Google の広告に関する情報とオプトアウト: https://adssettings.google.com

iOS の「設定 > プライバシーとセキュリティ > トラッキング」から、アプリによるトラッキングを制限できます。

### Amazon Web Services(AWS)

写真および認証情報の保存先は、**利用者ご自身が契約・管理する AWS アカウント** です。当該データの管理主体は利用者自身となります。AWS の取り扱いについては AWS のプライバシー通知(https://aws.amazon.com/jp/privacy/)をご参照ください。

## 広告について

本アプリでは、画面下部のバナー広告と、アップロード開始時に表示される動画(リワード)広告を配信します。広告は Google AdMob を通じて提供されます。

## 子どものプライバシー

本アプリは 13 歳未満の子どもを対象としていません。13 歳未満の利用者から意図的に情報を収集することはありません。

## データの保管と削除

- 写真・アカウント情報は利用者ご自身の AWS 環境に保存されており、**利用者自身が AWS の管理画面から削除・管理** できます。
- 端末内の設定・履歴は、本アプリを削除することで消去されます。
- アカウントの削除を希望する場合は、利用者ご自身の AWS の Cognito 管理画面から該当ユーザーを削除できます。

本ポリシーは、必要に応じて改定されることがあります。重要な変更がある場合は、本ページ上で告知します。
